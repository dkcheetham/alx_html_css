# CSS Advanced Project: Styling Web Pages

## Project Overview
This project is focused on applying advanced CSS concepts to style a webpage. The objective is to design a visually appealing and functional webpage based on a given Figma design.

## Features
- Responsive and interactive layout
- Custom fonts (`Source Sans Pro` and `Spin Cycle OT`)
- Advanced CSS concepts like Flexbox and Grid
- Styled buttons, headers, and footers
- W3C-compliant HTML and CSS

## Learning Objectives
By completing this project, you will learn:
- How to apply CSS styles effectively.
- The importance of specificity in CSS.
- Best practices for responsive web design.
- How to compute CSS specificity values.

## Tools and Resources
- HTML5
- CSS3
- Figma Design File
- [W3C Validator](https://validator.w3.org/) for code validation.

## Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a browser to view the webpage.
3. Edit the `style.css` file to customize the styles.
