# README.md for Headphones Project

## Project Overview
This project involves creating a fully functional web page from scratch using only HTML and CSS. The goal is to replicate the design provided in the Figma file while adhering to the principles of accessibility and responsive design. No external CSS frameworks or JavaScript are permitted.

---

## Requirements

- **No External CSS Frameworks**: You cannot use frameworks like Bootstrap.
- **No JavaScript**: The web page must be implemented without using JavaScript.
- **Responsive Design**:
  - Mobile version must activate when the screen width is 480px or less.
  - Max width of the content is set to 1000px, centered on the page.
- **Interactions**:
  - Links hover/active color: `#FF6565`
  - Button hover/active: `opacity: 0.9`
- **Figma Details**:
  - Use the [Figma project](<insert-link-here>) and duplicate it to your drafts.
  - Fonts required: `Source Sans Pro` and `Spin Cycle OT`. Download and install them if not available on your system.
  - Float values in Figma can be rounded as needed.

## Implementation Steps

1. **Understand the Design**:
   - Familiarize yourself with the Figma file.
   - Take note of font sizes, colors, margins, padding, and other styling details.

2. **Set Up the Project**:
   - Create an `index.html` file for the web page structure.
   - Create a `styles/style.css` file for CSS.
   - Organize assets like images and fonts in the `assets` directory.

3. **Build the HTML Structure**:
   - Use semantic HTML5 elements for better accessibility.
   - Follow the design structure in the Figma file.

4. **Style the Page with CSS**:
   - Implement styles from the Figma design.
   - Ensure responsiveness using media queries.
   - Apply hover and active states for links and buttons.

5. **Test for Responsiveness**:
   - Adjust the layout to switch to the mobile version at 480px or less.
   - Test the page in different screen sizes and devices.

6. **Final Review**:
   - Validate HTML and CSS using [W3C validators](https://validator.w3.org/).
   - Check for accessibility issues.
   - Confirm that the page matches the Figma design.
